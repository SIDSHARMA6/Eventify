# 🔐 User Permissions

## 👑 Admin
- ✅ Create, edit, delete ANY event
- ✅ View all events (including hidden)
- ✅ Manage locations (add, edit, delete, reorder)
- ✅ View all creators
- ✅ View creator statistics and summaries
- ✅ Scan QR codes for check-in
- ✅ View check-in history
- ✅ Delete tickets
- ✅ Access admin dashboard
- ✅ Access creator dashboard

## 🎨 Creator
- ✅ Create, edit, delete OWN events only
- ✅ View own events
- ✅ Hide/show own events
- ✅ Duplicate own events
- ✅ View event statistics
- ✅ Scan QR codes for check-in
- ✅ View check-in history
- ❌ Cannot delete tickets
- ❌ Cannot manage locations
- ❌ Cannot view other creators
- ❌ Cannot access admin dashboard

## 👤 User (Anonymous)
- ✅ View public events
- ✅ Book tickets (one per event per device)
- ✅ View own tickets
- ✅ Cancel own tickets
- ❌ No login required
- ❌ Cannot create events
- ❌ Cannot access admin/creator dashboards
